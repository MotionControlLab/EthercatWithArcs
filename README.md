# EthercatWithArcs

## setup

```sh
git clone https://github.com/MotionControlLab/EthercatWithArcs.git --recursive
cd ./EthercatWithArcs/master
```

## build & run

ARCS オンライン

```sh
sudo ./run.sh
```

ARCS オフライン

```sh
sudo ./run.sh offline
```
